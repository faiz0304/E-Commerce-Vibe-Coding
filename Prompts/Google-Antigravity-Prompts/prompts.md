## Prompt-2:

```prompt-txt
Build a modern and visually appealing frontend for an e-commerce website.

The website should have 5 main pages: Home, Shop, Product Details, Cart, and Checkout.

Design it like a modern online store with a hero section, featured products, product cards, and a clean shopping experience. Add smooth animations, hover effects, and responsive design so it looks good on both desktop and mobile.

The site should support around 20 typical e-commerce features such as browsing products, viewing product details, adding items to cart, updating quantities, removing products, and completing a simple checkout flow.

Keep everything lightweight and fast to generate. Use HTML, CSS, and JavaScript with a clean structure and modern UI so the final result looks like a real small online store.
```
