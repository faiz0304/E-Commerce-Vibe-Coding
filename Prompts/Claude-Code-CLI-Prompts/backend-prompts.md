## Prompt-1:

```prompt-txt
Build a lightweight backend for a small e-commerce website. The goal is to quickly generate a working backend that powers a simple online store.

The website should support around 20 basic e-commerce features such as product listing, product details, search, add to cart, remove from cart, simple checkout flow, order history, and basic user authentication.

Keep the backend simple and fast to generate. Instead of using a full database, store data using JSON or markdown files so everything stays lightweight and easy to run.

Create clean API endpoints that the frontend can use for products, cart, and orders. Organize the project structure in a clear way and include sample product data.

The backend should be easy to run locally and designed mainly for a demo project showing how quickly an AI can build a functional e-commerce system.
```
